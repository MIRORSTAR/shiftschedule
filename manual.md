# ShiftSchedule — Полная инструкция пользователя

---

## Что такое ShiftSchedule

ShiftSchedule — приложение для людей с посменной работой. Оно автоматически рассчитывает ваш рабочий календарь, напоминает о сменах будильником, показывает погоду на рабочие дни и позволяет синхронизировать график с партнёром.

---

## Первый запуск

### 1. Выбор типа графика

При первом открытии откройте **Настройки → График** и выберите один из вариантов:

| График | Описание |
|---|---|
| 1/1 | Один день работа — один выходной |
| 1/2 | Один рабочий — два выходных |
| 1/3 | Один рабочий — три выходных |
| 2/1 | Два рабочих — один выходной |
| 2/2 | Два рабочих — два выходных |
| 2/4 | Два рабочих — четыре выходных |
| 3/1 | Три рабочих — один выходной |
| 3/2 | Три рабочих — два выходных |
| 3/3 | Три рабочих — три выходных |
| 4/2 | Четыре рабочих — два выходных |
| 4/4 | Четыре рабочих — четыре выходных |
| 5/2 | Пятидневка |
| День/Ночь | Смешанный график с дневными и ночными сменами |

### 2. Установка даты начала цикла

После выбора графика нажмите **"Установить дату начала"** и укажите день, с которого начинается ваш рабочий цикл. Приложение само рассчитает все последующие смены.

## Календарь

### Что видно на экране

- **Синие ячейки** — рабочие дни
- **Серые ячейки** — выходные
- **Обведённая ячейка** — сегодня
- **Маленькая луна** в углу ячейки — ночная смена
- **Тёплые цвета** — особые статусы (подработка, отпуск, больничный)

### Смена статуса дня вручную

Нажмите на любой день — откроется карточка дня. Выберите нужный статус:

- **Работа** — обычная рабочая смена
- **Подработка** — дополнительный рабочий день
- **Выходной** — не рабочий день
- **Отпуск** — отпускной день (отменяет будильник)
- **Больничный** — больничный день (отменяет будильник)

Ручные изменения сохраняются и не сбрасываются при смене шаблона.

### Заметки к дням

В карточке дня есть поле для заметки. Текст заметки отображается под датой в списке смен.

---

## Список смен

Под календарём находится список предстоящих смен. В нём видно:

- **День недели и дата**
- **Иконка погоды** и температура (если включена погода)
- **Значок будильника** — если на этот день установлен будильник
- **Бейдж статуса** — Работа / Ночь / Выходной / Подработка / Отпуск / Больничный

Нажмите на строку в списке — откроется карточка этого дня.

## График День / Ночь

Для тех, у кого чередуются дневные и ночные смены.

### Как настроить

1. **Настройки → График → День / Ночь**
2. Добавьте смены кнопкой **"Добавить смену"**
3. Для каждой смены выберите тип: **День**, **Ночь** или **Выходной**
4. Выходные в конце цикла добавляются автоматически
5. Нажмите **"Применить"**

### Как это работает

Цикл повторяется с даты начала, которую вы указали. Приложение автоматически чередует дневные и ночные смены по заданной схеме. Ночные смены отмечаются значком луны в ячейке и в бейдже.

## Будильники

### Включение будильников

1. **Настройки → Будильники**
2. Выберите **время будильника** (за сколько минут до начала смены)
3. Нажмите **"Установить будильники"**
4. Разрешите приложению отправлять уведомления (система спросит автоматически)

### Как работают будильники

- Будильник срабатывает **за выбранное время** до начала смены
- Уведомление появляется на экране с кнопкой **"Закрыть"**
- Если день отмечен как Отпуск или Больничный — будильник на этот день автоматически отменяется
- Если вы меняете статус дня — будильники пересчитываются автоматически

### Настройка времени предупреждения

Нажмите на число минут (кнопки **−** и **+**) или тапните по цифре и введите вручную. Диапазон: от 5 до 240 минут.

## Погода

### Включение

**Настройки → Погода** → выберите режим:

- **Не показывать** — погода отключена
- **Только рабочие дни** — погода отображается только для смен
- **Все дни** — погода для каждого дня в списке

### Как это работает

Приложение запрашивает разрешение на определение местоположения и загружает прогноз погоды на ближайшие дни. В списке смен рядом с датой появляется иконка погоды и температура.

## Синхронизация с партнёром

Позволяет сравнивать свой график с графиком другого человека — чтобы планировать совместные выходные.

### Как поделиться своим графиком

1. **Настройки → Синхронизация → Войти** (Google-аккаунт)
2. После входа нажмите **"Загрузить мой график"**
3. Получите **код для партнёра** (6 символов) и отправьте его партнёру

### Как добавить партнёра

1. **Настройки → Синхронизация → Добавить партнёра**
2. Введите код, который партнёр вам прислал
3. График партнёра отобразится в вашем приложении

### Что видно после подключения

В списке смен под каждым днём появится строка: работает ли партнёр в этот день или тоже отдыхает.

---

## Синхронизация с календарём устройства

Переносит ваши смены в стандартный Календарь Android.

### Как настроить

1. **Настройки → Синхронизация → Синхронизация с устройством**
2. Включите переключатель
3. Выберите нужный календарь из списка
4. Выберите режим: **Только экспорт** или **Двусторонний**
5. Нажмите **"Синхронизировать"**

В двустороннем режиме события из вашего календаря (встречи, события) отображаются в списке смен прямо под датой.

### Светлая / Тёмная / Системная

**Настройки → Тема** — выберите один из трёх режимов. «Системная» следует настройкам вашего телефона.

Внутри каждой темы доступны 8 стилей оформления:

- **Graphite Bloom** — тёмно-серый с синим акцентом

- **iOS** — минималистичный светлый
- **Aurora** — глубокий синий с фиолетовым
- **Nordic** — холодный северный стиль

## Часто задаваемые вопросы

**Я сменил телефон. Как перенести график?**
Войдите в тот же Google-аккаунт → Настройки → Синхронизация → Скачать график с облака.

**График сбился — что делать?**
Зайдите в Настройки → График → проверьте дату начала цикла. Если нужно — установите новую.

**Будильник не срабатывает**
Проверьте: разрешение на точные будильники (Настройки телефона → Приложения → ShiftSchedule → Разрешения). На Android 12+ приложение попросит его при первом включении.

**Иконки не отображаются**
Обновите приложение до версии 1.3.1 и выше — эта проблема исправлена.

**Можно ли пользоваться без интернета?**
Да. Основной функционал (календарь, будильники, ручные статусы) работает полностью офлайн. Интернет нужен только для погоды, синхронизации с облаком и проверки обновлений.
