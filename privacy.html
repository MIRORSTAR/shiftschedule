Политика конфиденциальности

Последнее обновление: 21 апреля 2026 г.

1. Общие положения

Настоящая политика конфиденциальности описывает, как приложение ShiftSchedule («мы», «приложение») собирает, использует и защищает информацию пользователей.

2. Какие данные мы собираем

Данные, хранимые локально на устройстве:

  • Настройки графика смен (якорная дата, цикл, статусы дней)
  • Заметки к дням
  • Настройки приложения (тема, язык, стиль)
  • Настройки уведомлений

Все эти данные хранятся только на вашем устройстве и не передаются третьим лицам.

Данные при использовании облачной синхронизации (PRO):

  • Адрес электронной почты и зашифрованный пароль (через Firebase Authentication)
  • Данные графика смен (те же, что хранятся локально) — для синхронизации между устройствами

3. Геолокация

Приложение может запрашивать разрешение на геолокацию для отображения прогноза погоды. Координаты используются только для запроса погоды и не сохраняются на серверах приложения.

4. Доступ к календарю

При включении синхронизации с календарём приложение запрашивает доступ на чтение и запись событий в системный календарь устройства. Эти данные используются только для экспорта смен и не передаются третьим лицам.

5. Уведомления

Приложение использует локальные уведомления для напоминания о предстоящих сменах. Уведомления не содержат персональных данных и не передаются на внешние серверы.

6. Сторонние сервисы

  • Firebase (Google) — используется для аутентификации и хранения данных при облачной синхронизации (только PRO). Политика конфиденциальности Firebase
  • Open-Meteo — используется для получения прогноза погоды. Данные передаются только координаты местоположения. Условия использования Open-Meteo

7. Хранение данных

Локальные данные хранятся на вашем устройстве и удаляются при удалении приложения. Данные облачной синхронизации хранятся в Firebase и могут быть удалены по запросу пользователя.

8. Права пользователей

Вы можете в любой момент:

  • Отключить геолокацию в настройках телефона
  • Отключить доступ к календарю в настройках телефона
  • Удалить аккаунт облачной синхронизации через раздел «Синхронизация» в настройках приложения
  • Запросить удаление всех данных, написав на mirorstar@yandex.ru

9. Дети

Приложение не предназначено для детей до 13 лет и не собирает намеренно данные лиц до 13 лет.

10. Изменения политики

Мы можем обновлять настоящую политику конфиденциальности. Актуальная версия всегда доступна по адресу: mirorstar.github.io/shiftschedule/privacy.html

11. Контакты

По вопросам конфиденциальности: mirorstar@yandex.ru

Privacy Policy

Last updated: April 21, 2026

1. General

This privacy policy describes how the ShiftSchedule application ("we", "app") collects, uses and protects user information.

2. Data we collect

Data stored locally on the device:

  • Shift schedule settings (anchor date, cycle, day statuses)
  • Day notes
  • App settings (theme, language, style)
  • Notification settings

All this data is stored only on your device and is not shared with third parties.

Data when using cloud sync (PRO):

  • Email address and encrypted password (via Firebase Authentication)
  • Schedule data — for syncing between devices

3. Location

The app may request location permission to show weather forecasts. Coordinates are only used to fetch weather data and are not stored on our servers.

4. Calendar access

When calendar sync is enabled, the app requests read/write access to the device calendar. This data is only used to export shifts and is not shared with third parties.

5. Third-party services

  • Firebase (Google) — used for authentication and data storage for cloud sync (PRO only). Firebase Privacy Policy
  • Open-Meteo — used for weather forecasts. Only location coordinates are shared. Open-Meteo Terms

6. Contact

For privacy inquiries: mirorstar@yandex.ru
